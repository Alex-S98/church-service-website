# Church Sunday Service Website

A simple website for publishing your Sunday schedule, worship songs and lyrics, announcements, and church links. It uses plain HTML, CSS, and JavaScript, so it is free to host on Vercel and needs no database.

## Update the website each week

Open **data.js** and change:

- Church name, address, and welcome message
- Sunday date, time, sermon, speaker, and schedule
- Song names, artists, keys, and lyrics
- Announcements
- YouTube, Instagram, giving, and contact links

Only edit the text inside quotation marks. Keep commas and brackets in place.

> Important: publish song lyrics only when your church has permission or the song is in the public domain. Many modern worship songs are copyrighted and may require suitable church copyright/streaming licences.

## Preview on your computer

Double-click **index.html**. It will open in your browser.

## Put it on GitHub

1. Sign in at https://github.com and select **New repository**.
2. Name it **church-service-website** and create it.
3. Select **uploading an existing file**.
4. Upload all the files from this folder.
5. Select **Commit changes**.

## Publish on Vercel

1. Sign in at https://vercel.com using GitHub.
2. Select **Add New → Project**.
3. Import your **church-service-website** repository.
4. Leave the framework as **Other** and the build settings empty.
5. Select **Deploy**.

Vercel gives you a public address. Whenever you edit **data.js** on GitHub and commit it, Vercel automatically updates the live website.

## Use your own domain (optional)

In Vercel, open the project and go to **Settings → Domains**.
